==========================================
HEIST WIDGET - IMAGES FOLDER
==========================================

This folder is for any images used by the heist widget.

Currently empty - add images here as needed.

Examples:
- heist-icon.png
- background.jpg
- loading-spinner.gif

To use in CSS:
.heist-icon {
    background-image: url('images/icon.png');
}

To use in JS:
const BASE_URL = 'https://cdn.noticemesenpai.studio/heist/';
const iconPath = BASE_URL + 'images/icon.png';
